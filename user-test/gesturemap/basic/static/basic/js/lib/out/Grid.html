    Type Definitions

    Feedforward
    scene.js, line 343

    Represents the feedforward for the ideal execution of a gesture.
    Properties:
    Name Type Description
    line Konva.Line the line ( Konva.Line) representing the feedback.
    points Array.<Input.Point2D> the points of the polyline representing the ideal gesture execution.
    parts number the number of sub-parts in the ideal execution of the gesture.
    name string the name identifying the specific gesture. For the user test, they are triangle, square and delete.
    Example
    {
  line: new Konva.Line({
      points: [],
      stroke: '#008000',
      strokeWidth: 6,
      lineCap: 'round',
      lineJoin: 'round'
  }),
  points: [
      {x: 0, y: 100},
      {x: 100, y: 100},
      {x: 100, y: 0},
      {x: 0, y: 0}
  ],
  parts: 4,
  name: 'square'
};

    FeedforwardState
    scene.js, line 500

    Represents an update for the feedforward representation.
    Properties:
    Name Type Description
    name string the name of the gesture to update
    part number most likely part that the user is currently completing
    probability number the confidence associated by the recognition algorithm to the gesture part
    Example
    // we are 66% confident that the user is currently performing the third
// side of the square. The part counting starts from 0.
var squareUpdate = {name: 'square', part: 2, probability: 0.66}
